# 🐍 Simple Snake Game

A classic, browser-based **Snake Game** built with **React**. This project showcases state management, game loop handling, grid-based movement, and responsive collision detection in a clean React component architecture.

---

## ✨ Features

* **Classic Gameplay:** Navigate the snake to eat food, grow in length, and score points.
* **Grid-Based Rendering:** Precise collision checking for wall boundaries and self-intersections.
* **Responsive Controls:** Intuitive keyboard navigation (Arrow keys / WASD).
* **Score Tracking:** Real-time score counter updating with every food item consumed.
* **Restart & Reset:** Instant restart flow upon game over without needing a browser reload.

---

## 🛠️ Tech Stack

* **Frontend Library:** React.js
* **Language:** JavaScript (ES6+)
* **Styling:** CSS3

## 🚀 Getting Started

Follow these steps to run the game locally on your machine:

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your computer.

### Installation

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/ericksonvilleta/Simple-Snake-Game.git](https://github.com/ericksonvilleta/Simple-Snake-Game.git)
   ```
2. **Navigate to the project directory:**
   ```bash
   cd Simple-Snake-Game
   ```
3. **Install dependencies:**
   ```bash
   npm install
   ```
4. **Start the development server:**
   ```bash
   npm start
   ```
5. Open your browser and visit `http://localhost:3000` to play!

---

## 🎮 How to Play

1. Use the **Arrow Keys** (or **W, A, S, D**) to control the snake's direction.
2. Guide the snake toward the food target on the grid.
3. Each food eaten increases the snake's body length and boosts your score.
4. Avoid running into the grid borders or colliding with your own body.
